Fourth step: What the Path?

Ok, you've got all the packages, but your computer doesn't know where to look.
If you try to run a brew command right now, it will say brew command not found
You need to copy those lines in your .bashrc
export PATH="$HOME/.linuxbrew/bin:$PATH" export MANPATH="$HOME/.linuxbrew/share/man:$MANPATH" export INFOPATH="$HOME/.linuxbrew/share/info:$INFOPATH"
