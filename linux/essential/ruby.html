Ruby & Rvm Set Up

SHARE

First step: Install Ruby dependencies

Update your apps sudo apt-get update Install the dependencies: sudo apt-get install git-core curl zlib1g-dev build-essential libssl-dev libreadline-dev libyaml-dev libsqlite3-dev sqlite3 libxml2-dev libxslt1-dev libcurl4-openssl-dev python-software-properties libffi-dev
Type your password and hit Return

Next Page

Second step: Say Yes!

The terminal will ask you to validate the space that will be used
Write Y
And hit Return

Next Page

Third step: Is it finish?

You can think that it is finish because nothing is happenning on the screen
Wait until you see this screen

Next Page

Fourth step: Install RVM dependencies

Open your terminal again and type
sudo apt-get install libgdbm-dev libncurses5-dev automake libtool bison libffi-dev
The terminal will ask you to validate the space that will be used
Write Y
And hit Return

Next Page

Fifth step: Curl RVM

Curl the last stable version of RVM
curl -L https://get.rvm.io | bash -s stable

Next Page

Sixth step: Source it!

Use the source command to read & execute the content of /rvm
source ~/.rvm/scripts/rvm
Or if it is not working
source /home/YOUR COMPUTER NAME/.rvm/scripts/rvm

Next Page

Seventh step: Finally install ruby !

Run
rvm install 2.2.2

Next Page

Eight step: Use it!

Say to your computer which version it should use as a default version.
rvm use 2.2.2 --default
Now, check if you have the right version and no error messages
ruby -v

Next Page

Nineth step: Ready to find some gem?

Now tell to Rubygems to not install the documentation locally for each package.
echo "gem: --no-ri --no-rdoc" > ~/.gemrc Install the great bundler with: gem install bundler
