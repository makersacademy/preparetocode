First step: add a new repo and install psql

Run:
sudo sh -c "echo 'deb http://apt.postgresql.org/pub/repos/apt/ precise-pgdg main' > /etc/apt/sources.list.d/pgdg.list" Then:
wget --quiet -O - http://apt.postgresql.org/pub/repos/apt/ACCC4CF8.asc | sudo apt-key add -
Then: sudo apt-get update

Next Page

Second step: Install the common packages

Run:
sudo apt-get install postgresql-common
Write Y
And hit Return

Third step: Open the DMG

Run:
sudo apt-get install postgresql-9.3 libpq-dev
Write Y
And hit Return

Fourth step: Create yourself !

Postgre does not come with a default user so,
In order to be able to access the db and get some work done
Run:
sudo -u postgres createuser YOURNAME -s

Next Page

Fifth step: Create your password

Connect to psql with:
sudo -u postgres psql
Once your inside PSQL/SQL you cqn run SQL code so:
\password YOURNAME
Type your password two times
quit
