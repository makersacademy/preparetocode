Introduction: What's the toolbelt?

The Heroku toolbelt is a package that will allows you
to have the Heroku Command Line up and runing.
Using this you can interact directly with Heroku from the command line.

Let's start the setup

First step: Install the toolbelt

Go to this website:
https://toolbelt.heroku.com/osx
Download the package

Next Page

Second step: Open the package

Click on Open

Next Page

Third step: Continue the install

Click on Continue

Next Page

Fourth step: Put your password and launch the installation

Fill the input with your password
And click on Install Software

Next Page

Done!

That's it you are set up for Heroku, Foreman & Git now
