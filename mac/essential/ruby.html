First step: Install Ruby Version Manager (RVM)

Open your terminal and paste this line in: \curl -L https://get.rvm.io | bash -s stable
Basically RVM helps you to manage the different versions of ruby

Now, quit out of your terminal (CMD + Q)

Next Page

Second step: Check if everything goes well

Open your terminal again and type
type rvm | head -n 1
You should see a message saying
rvm is a function

Next Page

Third step: Finally we'll install Ruby

To install ruby, run this command: rvm use ruby --install --default
It might ask you for your password
Then if you type: ruby -v
If you get: ruby 2.2.1 or +
and your computer now speaks ruby
